## Internal Linking Structure

```
Home (/)
  ├─ For Sale (/properties/for-sale)
  │   ├─ Land (/properties/land)
  │   ├─ Houses (/properties/houses)
  │   ├─ Offices (/properties/offices)
  │   └─ By Area (East Legon, Tema, Kasoa, etc.)
  │
  ├─ For Rent (/properties/for-rent)
  │   ├─ Apartments (/properties/apartments)
  │   ├─ Houses (/properties/houses/rent)
  │   ├─ Furnished (/properties/furnished)
  │   └─ Short-term (/properties/short-term)
  │
  ├─ Guides & Resources
  │   ├─ How to Buy (/guides/how-to-buy)
  │   ├─ How to Rent (/guides/how-to-rent)
  │   ├─ Investing 101 (/guides/investing)
  │   └─ FAQ (/faq)
  │
  ├─ Agents (/agents)
  │   └─ [Individual Agent Profile]
  │
  └─ About/Contact/Terms (/about, /contact, /terms)
```

**Example Linking:**
- Home → links to "Browse Properties", "Learn How to Buy"
- For Sale page → links to "View Investment Guide", "Contact Agent"
- How-to-Buy guide → links to "View Properties", "Get Financing Info"
- Individual property → links to "Similar Properties", "Agent Profile", "Area Guide"

## SEO Checklist

- [ ] All pages have unique title tags (50–60 chars)
- [ ] All pages have unique meta descriptions (150–160 chars)
- [ ] Canonical URL set on each page
- [ ] OG image set for social sharing
- [ ] H1 present on each page (1 per page)
- [ ] H2/H3 hierarchy follows outline
- [ ] Internal links use descriptive anchor text
- [ ] Images have alt text (property, neighborhood photos)
- [ ] JSON-LD schema valid (test with Schema.org validator)
- [ ] Sitemap.xml includes all pages
- [ ] robots.txt allows crawling
- [ ] Mobile responsive (test with Mobile-Friendly Test)
- [ ] Page speed <3s (test with PageSpeed Insights)
- [ ] No duplicate content across pages
- [ ] 404 page is custom and helpful
- [ ] Breadcrumbs visible and marked with schema

## Measurement & Optimization

**Metrics to Track (Google Analytics):**
- Organic traffic by page
- Bounce rate per page (target <50%)
- Avg. session duration (target >2 min)
- Conversion rate (leads/calls/inquiries)
- Click-through rate from SERP (target >2%)

**Tools:**
- Google Search Console: monitor keywords, CTR, impressions
- Google Analytics 4: track behavior, conversion funnels
- Google PageSpeed Insights: monitor Core Web Vitals
- SEMrush/Ahrefs: competitor analysis, keyword research

**Optimization Cycles:**
- Monthly: review analytics, adjust content
- Quarterly: competitive analysis, update keyword targets
- Semi-annually: technical SEO audit, link building assessment
